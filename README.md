# SailsJs-PoC
Introduction to SailsJS

## Setup

### To install Sails:
´$ npm install sails -g´

### To generate a new app, just cd into the directory where you want it to be, and type:

´$ sails new test-project´

### Create user entity:
´$ sails generate api users´

### To take a look at your new Sails app:
´$ cd test-project´

### Run server
´$ sails lift´
